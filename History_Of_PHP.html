When was PHP created?

PHP was first created in 1994 by Danish Canadian programmer Rasmus Lerdorf. Originally, he used it for recording online visits to his resume that he had online. He name this the "Personal Home Page Tools," which is also referred to as "PHP Tools."

The General History of the language

  • 1995

    In June of 1995, Rasmus publicly released the source code to the public, which he users permission to improve the code itself.

    In September of 1995, Rasmus decided to expand and grow the PHP language, and implemented an update to the tools, which is referred as FI, which is short for "Forms Interpeter." It has new implemented features like "Perl-like" variables, auto interpetation of the form variables, and HTML syntax.

    In October of 1995, Rasmus rewritten the PHP code to have a fair resemblence to the C programming language structure-wise.

  • 1996

    In April of 1996, PHP has another complete code overhaul, and named it PHP/FI, which included a built-in support for DBM (in computing it is just a way to access data), mSQL, Postgres95, databases, cookies, and more important website related features.

  • 1997

    In November of 1997, the parsing engine of PHP was rewritten.

  • PHP 3 (1998)

    In June of 1998, PHP 3.0 was announced by the PHP development team, which is also installed on 70,000 domains around the world, and was installed aboud 10% over the web servers that is on the internet.

  • PHP 4 (1998-2000)

    In the winter of 1998, Andi Gutmans & Zeev Suraski started a rewrite of PHP. In 1999, the new engine, "Zend Engine", which is a merge between Andi and Zeev's name, was implemented into PHP 4.0, which has new features for supporting web servers, HTTP sessions, output buffering, and more secure ways of managing user input while also having many new language constructions.

  • PHP 5 (2004)

    In July of 2004, PHP 5 was released. New features it has was the Zend Engine 2.0 a new object model, and more.

  • PHP 6, 5.3, & 5.4 (between 5 and 7)

    Though version 6 of PHP was expected to release, there were many development challenges, and was not worked on. However, features of PHP 6 was implemented to 5.3 and 5.4 instead.

  • PHP 7 (2015)

    In 2015, PHP was released. New features that it has were an update to the Zend Engine, which is the 3.0 and had a performance boost, scalar type declarations, & return type declarations.

  • PHP 8 (2020)

    In 2020, PHP 8 was released. New features that it has were the Just-In-Time compiler, which optimizes on what type of compilation works based on certain lines of code is for performance, Union types, Attributes, & Match expresstion.

What types of problems is the language used for?
