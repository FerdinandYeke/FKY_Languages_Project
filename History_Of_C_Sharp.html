When was C# created, and by who?

C sharp was first created in 2002. C sharp was created by Mr. Andres Hejlsberg. C sharp was made to be generally used, as well as being an object-oriented programming language from the start. C# throughout the years was also being developed by Microsoft.

The General History of the language

  • C# v1.0 (2002)

    In January of 2002,C# was released. features that it had were Classes, Structs, Interfaces, Events, Properties, Delegates, Operators and expressions, Statements, Attributes, and more. It also came with .NET framework 1.0, and 1.1, while also having a CLR version 1.0 and Microsoft Visual Studio 2002.

  • C# v1.2 (2003)

    In April of 2003, C# v1.2 was released. This version had new improvments to the language, like having "Dispose", "IEnumerator", and "IDisposable", which are some code generated in a foreach loop. C# also came with Visual Studio .NET 2003.

  • C# v2.0 (2005)

    In November of 2005, C# v2.0 was released. Features that it have are Generics, Partial types, Anonymous methods, Nullable value types, Iterators, Covariance and Contravariance. C# v2.0 also came with Visual Studio 2005.

  • C# v3.0 (2007)

    In November of 2007, C# v3.0 was released, along with Visual Studio 2008 & .NET Framework v3.5. This is the version that really made C# a serious programming language. Features that it have are Auto-implemented properites, Anonymous types, Query expressions, Expression tress, Extension methods and so much more.

  • C# v4.0 (2010)

    In April of 2010, C# v4.0 was released, along with Visual Studio 2010. This version had new features implemented, which are Dynamic binding, Named/optional arguments, Generic covariant and contravariant, Embedded interop types, and more.

  • C# v5.0 (2012)

    In August of 2012, C# v5.0 was released with Visual Studio 2012. This version unlike the others was much more "focused" on the language itself. A major feature this had were implementing concepts like " asynce" and "await" model for "asynchronous" programming. Other major features this version had are Asynchorous memebers, Caller info attributes, Code Project: Caller Info Attributes in C# 5.0.

  • C# v6.0 (2015)

    In July of 2015, C# v6.0 was released, along with Visual Studio 2015. This version had new features implemented, which are Static imports, Exception filters, Auto-property initializers, Expression bodied member, Null propagator, and more.

  • C# v7.0 (2017)

    In March of 2017, C# v7.0 was released, along with Visual Studio 2017. This version had new features implemented, which are Out variables, Tuples and deconstruction, Pattern matching, local functions, Expanded expression bodied members, Ref locals, & Ref returns. Other features that it had were Discards, Binary Literals and Digit Seperators, & Throw expressions.

  • C# v7.1 (2017)

    In August of 2017, C# v7.1 was released. This version had new features implemented, which are asynce Main method, default literal expressions, Inferred tuple element names, & Patter matching on generic type parameters. C# also had new compiler behaviors, like with -refout and -refonly.

  • C# v7.2 (2017)

    In November of 2017, C# v7.2 was released. This version had small new features implemented, which are Non-trailing named arguments, private protected access modifier, Conditional ref expressions, and more.

  • C# v7.3 (2018)

    In May of 2018, C# v7.3 was released. This version had more new features implemented, which they help support safer code, while making it as "performant" as unsafe code.

  • C# v8.0 (2019)

    In September of 2019, C# v8.0 was released. This version unlike the others was the first big C# release that primarly focused on the .NET Core. New features that this language added were Readonly members, Patter matching enhancements, Using declarations, Static local functions, Disposable ref structs, Nullable reference types, and more.

  • C# v9 (2020)

    In November of 2020, C# v9 was released with .NET 5, which is the default language for any kind of assembly that targets it. This version had more new features implemented, which are Records, Init only setters, Top-level statments, Preformance and interop, Fit and Finish features, and more.

  • C# v10 (2021)

    In November of 2021, C# v10 was released. C# v10, adds new addtional features to the language, which were Record Structs, Improvements of structure types, Interpolated string handlers, global using directives, File-scoped namespace declaration, and much, much more.

  • C# v11 (2022)

    In November of 2022, C# v11 was released. C# adds new additional features to the language, which were Raw string literals, Generic math support, Generic attributes, UTF-8 string literals, Newlines in string interpolation expressions, List patterns, and much more.

  • C# v12 (2023)

    In November of 2023, C# v12 was released. Additional features that were added to the language were Primary constructors, Collection expressions, Inline Array, Optional parameters in lambda expressions, ref readonly parameters, Alias any type, & Experimental attribute.

What types of problems is the language used for?

As you seen and learned above, the C# language was a general language that can make programs, software, and web applications professionaly and efficently.

What is the current state of the language?

As the time of writing this, C# v12 is the current version of the programming language. It is still used to make programs/software and web applications. It is still an ongoing language that evolved with the times, and will continue to grow and evolve with other programming languages today and into the future.
